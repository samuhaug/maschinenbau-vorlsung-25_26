Schleifen: for & while

Messreihen verarbeiten, Spielrunden steuern, Daten durchlaufen

WS 2025/26  ·  DHBW Stuttgart  ·  Nachschlagewerk Python-Praxis

01

Das kannst du danach

📌 Nach dieser Einheit kannst du …

  • mit for und range() eine Aktion eine bestimmte Anzahl mal wiederholen.
  • mit for über Listen und Texte iterieren (z. B. alle Messwerte ausgeben).
  • mit while eine Schleife so lange laufen lassen, wie eine Bedingung gilt.
  • Endlosschleifen erkennen und durch korrekte Abbruchbedingungen vermeiden.
  • mit enumerate() gleichzeitig Index und Wert einer Liste durchlaufen.
02

Wann for, wann while?

for-Schleife

Wenn du weißt, wie oft etwas wiederholt werden soll – z. B. 10 Messwerte einlesen, alle Elemente einer Liste verarbeiten.

while-Schleife

Wenn du nicht weißt, wie oft – z. B. solange der Nutzer kein "ende" eingibt, solange Temperatur < Sollwert.

⚠ Endlosschleife vermeiden

Eine while-Schleife, deren Bedingung nie False wird, läuft ewig. Stelle sicher, dass sich irgendetwas in der Schleife verändert, das irgendwann die Bedingung auf False setzt.

03

Schleifen im Detail

for + range() – Zählschleife

range(start, stop, step) erzeugt eine Folge von Ganzzahlen. stop ist nicht enthalten (exklusive Ende).

for variable in range(stop): # 0 bis stop-1 for variable in range(start, stop): # start bis stop-1
# 5 Messwerte einlesen for i in range(5): wert = float(input(f"Messung {i+1}: ")) print(f" → {wert:.2f} °C") # Zähler mit Start und Schritt for sekunde in range(0, 60, 10): # 0, 10, 20, 30, 40, 50 print(f"t = {sekunde} s: Messung läuft...") # range() Merkhilfe: # range(5) → 0,1,2,3,4 # range(1,6) → 1,2,3,4,5 # range(0,10,2) → 0,2,4,6,8
for – über Liste / Text iterieren

Mit for x in liste durchläufst du jeden Wert direkt. Mit enumerate() erhältst du zusätzlich den Index.

messwerte = [23.1, 24.5, 22.8, 25.0, 23.7] # Direkt über Werte iterieren for wert in messwerte: print(f"Temperatur: {wert:.1f} °C") # Mit enumerate(): Index + Wert for i, wert in enumerate(messwerte): print(f"Messung {i+1}: {wert:.1f} °C") # Über String iterieren eingabe = "A3" for zeichen in eingabe: print(zeichen) # → A, dann 3
while – bedingte Schleife

while wiederholt den Block so lange, bis die Bedingung False ist. Typisch: auf Nutzereingabe warten, Prozess überwachen.

while bedingung: # läuft solange bedingung True ist
# Eingabe wiederholen bis gültig wert = -1 while wert < 0: wert = float(input("Positive Temperatur eingeben: ")) if wert < 0: print("Ungültig! Bitte positive Zahl.") # Prozess überwachen temp = 20.0 while temp < 80: print(f"Aufheizen: {temp:.1f} °C") temp += 5.0 # temp muss sich ändern! print("Betriebstemperatur erreicht.")

05

Quick-Reference

SyntaxBedeutung & Hinweis
for i in range(n):n mal wiederholen, i läuft von 0 bis n-1
for i in range(a,b):a bis b-1 (b ist exklusiv!)
for i in range(a,b,s):Mit Schrittweite s (kann auch negativ sein)
for x in liste:Jeden Wert in der Liste einmal verarbeiten
for i, x in enumerate(liste):Index i + Wert x gleichzeitig
while bedingung:Läuft solange Bedingung True – Variable muss sich ändern!
while True:Endlosschleife – nur mit break beenden
