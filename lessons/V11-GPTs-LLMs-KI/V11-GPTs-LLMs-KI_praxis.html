Funktionen im Detail

def + return – Grundaufbau

def definiert eine Funktion. return gibt einen Wert zurück. Ohne return gibt die Funktion None zurück.

def funktionsname(parameter): # Eingerückter Code-Block return ergebnis
# Funktion definieren def celsius_zu_kelvin(celsius): kelvin = celsius + 273.15 return kelvin # Funktion aufrufen temp_k = celsius_zu_kelvin(23.5) print(f"{temp_k:.2f} K") # → 296.65 K # Funktion mehrmals aufrufen – das ist der Vorteil! for t in [0, 20, 100]: print(f"{t} °C = {celsius_zu_kelvin(t):.2f} K")
Parameter & Standardwerte

Parameter mit Standardwert müssen beim Aufruf nicht angegeben werden – der Standardwert wird dann verwendet. Parameter ohne Standardwert sind Pflichtparameter.

def berechne_mittelwert(werte, einheit="°C"): mw = sum(werte) / len(werte) print(f"Mittelwert: {mw:.2f} {einheit}") return mw temps = [22.1, 24.5, 23.8] # Ohne optionalen Parameter – Standardwert "°C" gilt berechne_mittelwert(temps) # Mit eigenem Parameter – überschreibt Standardwert berechne_mittelwert(temps, "K")
Mehrere Rückgabewerte

Python kann mehrere Werte gleichzeitig zurückgeben – als Tupel. Das ist sehr praktisch für Auswertungsfunktionen.

def auswerten(messwerte): minimum = min(messwerte) maximum = max(messwerte) mittel = sum(messwerte) / len(messwerte) return minimum, maximum, mittel # gibt Tupel zurück werte = [22.1, 24.5, 87.3, 23.8] mn, mx, avg = auswerten(werte) # Tuple-Unpacking print(f"Min: {mn:.1f} Max: {mx:.1f} Ø: {avg:.1f}")

def + return für Spielfeld-Aufbau

# spiellogik.py def erstelle_spielfeld(): feld = [] for i in range(5): feld.append(["~"] * 5) return feld # gibt fertiges 5×5-Feld zurück def spielfeld_ausgeben(spielfeld): print(" A B C D E") for i, zeile in enumerate(spielfeld): print(f"{i+1} ", end="") print(" ".join(zeile)) # Verwendung im Hauptprogramm feld = erstelle_spielfeld() # Rückgabewert nutzen spielfeld_ausgeben(feld) # kein Rückgabewert nötig
