02

Was ist eine Variable?

Eine Variable ist ein benannter Speicherplatz im Arbeitsspeicher. Du kannst dir eine Variable wie ein beschriftetes Kästchen vorstellen: Du gibst ihm einen Namen und legst einen Wert hinein. Später kannst du den Wert jederzeit auslesen oder ändern.

In Python brauchst du kein spezielles Schlüsselwort wie var oder int – du weist einfach einen Wert zu, und Python erkennt den Typ automatisch.

Gute Variablennamen

Verwende beschreibende Namen in Kleinbuchstaben mit Unterstrichen: drehzahl_rpm, temperatur_celsius, messung_nr. Meide kryptische Abkürzungen wie x, tmp1, val.

03

Funktionen im Detail

print()

Gibt einen oder mehrere Werte auf der Konsole aus. Python fügt nach jeder Ausgabe automatisch einen Zeilenumbruch ein. Du kannst mehrere Werte durch Kommas trennen – Python fügt dann automatisch ein Leerzeichen dazwischen ein.

print(wert1, wert2, ..., sep=" ", end="\n")
wert1, wert2, … – Beliebig viele Werte, die ausgegeben werden sollen
sep= – Trennzeichen zwischen mehreren Werten (Standard: Leerzeichen)
end= – Zeichen am Ende der Ausgabe (Standard: Zeilenumbruch \n)
# Einfache Textausgabe print("Messung gestartet") # Variable ausgeben temperatur = 23.5 print("Temperatur:", temperatur) # → Temperatur: 23.5 # Mehrere Werte mit eigenem Trennzeichen print("T=", temperatur, "°C", sep="") # → T=23.5°C # Ohne Zeilenumbruch am Ende print("Schritt 1", end=" | ") print("Schritt 2") # → Schritt 1 | Schritt 2
input()

Hält das Programm an und wartet auf eine Tastatureingabe des Nutzers. Der eingegebene Text wird als Ergebnis zurückgegeben. Wichtig: input() liefert immer einen Text (String) – auch wenn der Nutzer eine Zahl eingibt! Für Berechnungen muss der Text erst umgewandelt werden.

eingabe = input("Aufforderungstext")
⚠ Achtung: input() liefert immer einen Text!

Auch wenn der Nutzer 42 eingibt, enthält die Variable den String "42", nicht die Zahl 42. Für Berechnungen muss man mit int() oder float() umwandeln.

# Texteingabe einlesen name = input("Dein Name: ") print("Hallo,", name) # Zahl einlesen und umwandeln eingabe = input("Temperatur in °C: ") temperatur = float(eingabe) # Text → Dezimalzahl print("Gespeichert:", temperatur, "°C") # Kurzschreibweise (alles in einer Zeile) drehzahl = int(input("Drehzahl in RPM: ")) print("RPM eingelesen:", drehzahl)
Variablen

Variablen speichern Werte unter einem Namen. In Python wird der Typ automatisch erkannt – du musst ihn nicht angeben. Ein Gleichheitszeichen = bedeutet in Python immer „weise zu", nicht „ist gleich".

variablenname = wert
# Zuweisung: links = Name, rechts = Wert temperatur = 23.5 # float: Dezimalzahl messung_nr = 1 # int: Ganzzahl einheit = "Grad Celsius"# str: Text sensor_aktiv = True # bool: Wahrheitswert # Wert ändern – einfach neu zuweisen messung_nr = 2 # überschreibt den alten Wert # Wert erhöhen (häufiges Muster) messung_nr = messung_nr + 1 # jetzt 3 messung_nr += 1 # Kurzschreibweise: jetzt 4 # Mehrere Zuweisungen in einer Zeile x, y, z = 0, 0, 0
Namensregeln

Variablennamen dürfen Buchstaben, Ziffern und _ enthalten – aber nicht mit einer Ziffer beginnen. Groß- und Kleinschreibung wird unterschieden: Temperatur und temperatur sind zwei verschiedene Variablen.

# Kommentare

Kommentare sind Erklärungen im Code, die Python komplett ignoriert. Sie beginnen mit dem Zeichen # und gelten bis zum Zeilenende. Gute Kommentare erklären das Warum, nicht das offensichtliche Was.

# Das ist ein Kommentar  ←  gilt bis Zeilenende
# ────────────────────────────────────────────────────── # Programm: Temperatur-Logger # Zweck: Messwert einlesen und ausgeben # ────────────────────────────────────────────────────── messung_nr = 1 # aktuelle Messung (beginnt bei 1) # Messung einlesen wert = float(input("Temperatur (°C): ")) print("Messung", messung_nr, ":", wert, "°C")
Wann kommentieren?

Kommentiere komplexe Formeln, nicht-offensichtliche Entscheidungen und den Zweck von Abschnitten. Nicht nötig: messung_nr = 1 # setzt messung_nr auf 1 – das liest man direkt aus dem Code.
