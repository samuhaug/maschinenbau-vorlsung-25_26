Diagramme mit matplotlib

Messdaten visualisieren: Linien-, Balken- und Streudiagramme

WS 2025/26  ·  DHBW Stuttgart  ·  Nachschlagewerk Python-Praxis

01

Das kannst du danach

📌 Nach dieser Einheit kannst du …

  • mit matplotlib.pyplot Diagramme aus Python heraus erstellen.
  • Liniendiagramme für Zeitreihen und Messdatenverläufe zeichnen.
  • Balkendiagramme für Vergleiche (z. B. Bauteile, Messstationen) erstellen.
  • Streudiagramme für Korrelationsanalysen einsetzen.
  • Achsen beschriften, Titel setzen, Legende hinzufügen und Diagramme speichern.
Installation

Falls matplotlib nicht installiert ist: Terminal öffnen und pip install matplotlib ausführen. In Jupyter: !pip install matplotlib

02

Die drei wichtigsten Diagrammtypen

📈
plt.plot()

Liniendiagramm – für Zeitreihen, Temperaturverlauf, kontinuierliche Messdaten

📊
plt.bar()

Balkendiagramm – für Kategorienvergleiche, Produktionsmengen, Ranking

🔵
plt.scatter()

Streudiagramm – für Korrelationen, Messwert-Verteilungen, Ausreißer erkennen

03

Diagramme im Detail

Grundstruktur jedes Diagramms

Jedes matplotlib-Diagramm folgt diesem Schema: importieren → Daten → Diagrammtyp → Beschriftung → anzeigen/speichern.

import matplotlib.pyplot as plt # 1. Daten zeiten = [0, 10, 20, 30, 40, 50] temps = [20.1, 25.3, 45.8, 72.1, 88.4, 92.0] # 2. Diagramm zeichnen plt.plot(zeiten, temps) # 3. Beschriftungen plt.xlabel("Zeit [s]") plt.ylabel("Temperatur [°C]") plt.title("Aufheizkurve Motor") # 4. Anzeigen (oder speichern) plt.savefig("aufheizung.png") # als Datei speichern plt.show() # Fenster öffnen
plt.plot() – Liniendiagramm erweitert
import matplotlib.pyplot as plt t = [0, 5, 10, 15, 20] sensor1 = [20, 35, 55, 72, 88] sensor2 = [18, 30, 50, 68, 84] # Farbe, Linienstil, Beschriftung für Legende plt.plot(t, sensor1, color="#E2001A", linewidth=2, label="Sensor 1") plt.plot(t, sensor2, color="#60A5FA", linestyle="--", label="Sensor 2") # Grenzlinie (Alarmgrenze) plt.axhline(y=80, color="orange", linestyle=":", label="Alarmgrenze") plt.xlabel("Zeit [min]") plt.ylabel("Temperatur [°C]") plt.title("Temperaturverlauf Bauteil XY") plt.legend() # Legende anzeigen plt.grid(True) # Gitternetz plt.show()
plt.bar() – Balkendiagramm
import matplotlib.pyplot as plt stationen = ["Station A", "Station B", "Station C", "Station D"] produktion = [142, 189, 97, 213] plt.bar(stationen, produktion, color="#E2001A") plt.xlabel("Fertigungsstation") plt.ylabel("Teile pro Schicht") plt.title("Produktionsvergleich KW 23") plt.savefig("produktion.png", dpi=150, bbox_inches="tight") plt.show()
plt.scatter() – Streudiagramm
import matplotlib.pyplot as plt # Korrelation: Drehzahl vs. Temperatur drehzahl = [800, 1000, 1200, 1500, 1800, 2000] temperatur = [45, 52, 61, 74, 83, 91 ] plt.scatter(drehzahl, temperatur, color="#E2001A", s=80) plt.xlabel("Drehzahl [RPM]") plt.ylabel("Lagertemperatur [°C]") plt.title("Drehzahl vs. Temperatur") plt.grid(True, alpha=0.3) plt.show()
