f-Strings & Ausgabeformatierung

Messwerte, Tabellen und Berichte professionell formatiert ausgeben

WS 2025/26  ·  DHBW Stuttgart  ·  Nachschlagewerk Python-Praxis

01

Das kannst du danach

📌 Nach dieser Einheit kannst du …

  • mit f-Strings Variablen direkt in Texte einbauen.
  • Zahlen auf eine bestimmte Anzahl Nachkommastellen formatieren.
  • Ausgaben in Spalten ausrichten – z. B. für Messtabellen.

02

Aufbau eines f-Strings

Ein f-String (formatted string literal) ist ein Text, in den du direkt Variablen oder Ausdrücke einbetten kannst. Er beginnt mit dem Buchstaben f vor dem Anführungszeichen. Alles in { } wird als Python-Ausdruck ausgewertet.

Das Ergebnis: Temp: 23.50 °C – der Wert der Variable temp wird auf 2 Nachkommastellen gerundet eingesetzt.

03

Formatierungsoptionen im Detail

f"...{variable}..."

Der einfachste f-String: du schreibst f vor den String und setzt Variablen oder Ausdrücke in { }. Python ersetzt die Klammern durch den Wert der Variablen.

f"Text {variable} weiterer Text"
drehzahl = 1450 einheit = "RPM" print(f"Drehzahl: {drehzahl} {einheit}") # → Drehzahl: 1450 RPM # Ausdruck direkt im f-String berechnen print(f"Doppelt: {drehzahl * 2} RPM") # → Doppelt: 2900 RPM
:.2f  –  Nachkommastellen

Mit :.2f (f für float) gibst du an, wie viele Nachkommastellen ausgegeben werden sollen. Python rundet den Wert dabei automatisch. Besonders nützlich bei Messwerten, die immer gleich viele Dezimalstellen haben sollen.

f"{wert:.2f}"  ← 2 Nachkommastellen
temperatur = 23.567891 druck = 1.0133 print(f"Temperatur: {temperatur:.1f} °C") # → 23.6 °C print(f"Temperatur: {temperatur:.3f} °C") # → 23.568 °C print(f"Druck: {druck:.4f} bar") # → 1.0133 bar # Ganze Zahl mit führenden Nullen nr = 7 print(f"Messung-{nr:03d}") # → Messung-007
:10  –  Spaltenbreite

Mit :10 reservierst du 10 Zeichen Platz für den Wert. So entstehen saubere Tabellen. Standardmäßig wird linksbündig (Text) bzw. rechtsbündig (Zahlen) ausgerichtet. Mit <, ^, > kannst du die Ausrichtung steuern.

{wert:<10} links  |  {wert:^10} mittig  |  {wert:>10} rechts
# Messtabelle mit gleichmäßigen Spalten print(f"{'Sensor':<15} {'Wert':>8} {'Einheit':<8}") print("-" * 35) print(f"{'Temperatur':<15} {23.5:>8.2f} {'°C':<8}") print(f"{'Drehzahl':<15} {1450:>8d} {'RPM':<8}") print(f"{'Druck':<15} {1.0133:>8.4f} {'bar':<8}") # → Sensor Wert Einheit # → ----------------------------------- # → Temperatur 23.50 °C # → Drehzahl 1450 RPM # → Druck 1.0133 bar
\n  \t  –  Sonderzeichen

Innerhalb von Strings haben bestimmte Zeichenkombinationen mit Backslash eine besondere Bedeutung. Sie heißen Escape-Sequenzen.

# Zeilenumbruch \n print("Zeile 1\nZeile 2\nZeile 3") # → Zeile 1 # → Zeile 2 # → Zeile 3 # Tabulator \t (für Ausrichtung) print("Sensor\tWert") print("Temp\t23.5") # Anführungszeichen im String: \" oder '...' print('Er sagte "Treffer!"') # Backslash selbst: \\ print("Pfad: C:\\Daten\\messung.csv")
Wann \t, wann Spaltenbreite?

Tabulatoren (\t) springen zur nächsten Tab-Position (normalerweise alle 8 Zeichen) – das Ergebnis hängt vom Terminal ab. Für saubere Tabellen in Berichten ist die Spaltenbreite mit :{breite} zuverlässiger.

05

Quick-Reference

Format-Angabe Beispiel Ergebnis / Bedeutung
f"{var}"f"T={temp}"Wert direkt einsetzen
f"{var:.2f}"f"{23.5678:.2f}"23.57 – 2 Nachkommastellen
f"{var:.0f}"f"{23.7:.0f}"24 – auf Ganzzahl runden
f"{var:d}"f"{42:d}"Ganzzahl (integer)
f"{var:03d}"f"{7:03d}"007 – 3-stellig mit führenden Nullen
f"{var:<10}"f"{'Temp':<10}"Temp – 10 Zeichen linksbündig
f"{var:>10}"f"{1450:>10}" 1450 – 10 Zeichen rechtsbündig
f"{var:^10}"f"{'OK':^10}" OK – 10 Zeichen zentriert
\nprint("A\nB")Zeilenumbruch im String
\tprint("A\tB")Tabulator (Abstandszeichen)
Praxis-Tipp: Messbericht ausgeben
sensoren = [("Temperatur", 23.567, "°C"), ("Drehzahl", 1450, "RPM"), ("Druck", 1.0133, "bar")] print(f"{'Sensor':<12} {'Wert':>10} Einheit") print("-" * 32) for name, wert, einheit in sensoren: print(f"{name:<12} {wert:>10.3f} {einheit}")
