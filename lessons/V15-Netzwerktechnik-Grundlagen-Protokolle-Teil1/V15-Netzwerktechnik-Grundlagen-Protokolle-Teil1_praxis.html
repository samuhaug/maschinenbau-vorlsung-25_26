Was ist CSV?

CSV (Comma-Separated Values) ist das universelle Format für Tabellendaten. Jede Zeile ist ein Datensatz, Spalten sind durch Kommas getrennt. Excel, MATLAB, Messtechnik-Software – alle können CSV lesen.

02

CSV-Format

Eine typische CSV-Datei aus einer Messung könnte so aussehen:

# messung.csv zeitpunkt,temperatur,drehzahl,status 0,22.1,800,OK 10,35.4,1200,OK 20,61.2,1500,OK 30,87.3,1800,ALARM 40,91.0,1800,ALARM

Die erste Zeile ist der Header (Spaltenbezeichnungen). Alle weiteren Zeilen sind Datensätze.

03

CSV verarbeiten

csv.reader – CSV lesen

Das csv-Modul ist Teil der Python-Standardbibliothek – kein pip install nötig. Es behandelt Sonderfälle automatisch (Anführungszeichen, Kommas in Texten).

import csv with open("messung.csv", "r", encoding="utf-8") as datei: leser = csv.reader(datei) header = next(leser) # erste Zeile = Header überspringen print("Spalten:", header) for zeile in leser: t = int(zeile[0]) temp = float(zeile[1]) rpm = int(zeile[2]) status = zeile[3] print(f"t={t}s {temp:.1f}°C {rpm}RPM [{status}]")
csv.DictReader – CSV mit Spaltennamen

DictReader liefert jede Zeile als Dictionary – du kannst die Spalten beim Namen ansprechen statt per Index. Deutlich lesbarer.

import csv temperaturen = [] with open("messung.csv", "r", encoding="utf-8") as datei: leser = csv.DictReader(datei) # Header automatisch for zeile in leser: temp = float(zeile["temperatur"]) # per Name! temperaturen.append(temp) print(f"Max: {max(temperaturen):.1f} °C") print(f"Ø: {sum(temperaturen)/len(temperaturen):.1f} °C")
CSV schreiben · JSON-Grundlage
import csv, json # CSV schreiben daten = [[0, 22.1], [10, 35.4], [20, 61.2]] with open("ausgabe.csv", "w", newline="", encoding="utf-8") as f: schreiber = csv.writer(f) schreiber.writerow(["zeit", "temperatur"]) # Header schreiber.writerows(daten) # Alle Zeilen # JSON lesen (z.B. Konfigurationsdatei) with open("config.json", "r") as f: config = json.load(f) print(config["alarm_limit"]) # Zugriff wie Dictionary
