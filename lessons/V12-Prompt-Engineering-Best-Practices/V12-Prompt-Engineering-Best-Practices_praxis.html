02

String-Methoden als Kette

String-Methoden können verkettet (chained) werden: das Ergebnis einer Methode wird direkt an die nächste übergeben.

# Verkettung: Leerzeichen entfernen + in Großbuchstaben eingabe = " a3 \n" bereinigt = eingabe.strip().upper() # → "A3"
03

Methoden im Detail

strip() · upper() · lower() · replace()
rohwert = " 23.5 °C \n" # strip(): Leerzeichen + Sonderzeichen am Rand entfernen print(rohwert.strip()) # → "23.5 °C" # upper() / lower(): Groß- / Kleinschreibung print("Alarm".upper()) # → "ALARM" print("SENSOR".lower()) # → "sensor" # replace(alt, neu): Text ersetzen zeile = "23.5,24.1,22.8" clean = zeile.replace(",", " ") # → "23.5 24.1 22.8" # Komma als Dezimaltrenner → Punkt (deutsches Zahlenformat) wert = "23,5".replace(",", ".") # → "23.5" temp = float(wert) # → 23.5
split() · join()

split() zerlegt einen String in eine Liste. join() verbindet eine Liste zu einem String – das Gegenteil.

# split(): String → Liste csv_zeile = "Temperatur,23.5,°C,OK" teile = csv_zeile.split(",") print(teile) # → ['Temperatur', '23.5', '°C', 'OK'] print(teile[1]) # → '23.5' wert = float(teile[1]) # → 23.5 (Zahl) # split() ohne Argument: trennt bei Leerzeichen satz = "Motor ist aktiv" print(satz.split()) # → ['Motor', 'ist', 'aktiv'] # join(): Liste → String felder = ["Sensor1", "22.1", "°C"] zeile = ",".join(felder) print(zeile) # → "Sensor1,22.1,°C"
startswith() · endswith() · in

Texte durchsuchen ohne reguläre Ausdrücke – einfach und praxisnah.

dateiname = "messung_2026_01.csv" meldung = "ALARM: Überhitzung Sensor 3" # startswith() / endswith() print(dateiname.startswith("messung")) # → True print(dateiname.endswith(".csv")) # → True # in-Operator: Text enthält Teilstring? print("ALARM" in meldung) # → True print("Sensor 3" in meldung) # → True # Praktisch: alle Alarm-Zeilen filtern log = ["INFO: Start", "ALARM: Temp", "INFO: OK"] for zeile in log: if zeile.startswith("ALARM"): print("⚠", zeile)
