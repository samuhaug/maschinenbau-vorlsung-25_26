Verzweigungen im Detail

if – einfache Bedingung

Der if-Block wird nur ausgeführt, wenn die Bedingung True ergibt. Wichtig: der eingerückte Block (4 Leerzeichen oder Tab) gehört zum if.

if bedingung: # dieser Code läuft nur wenn bedingung True ist
temperatur = 95.0 grenzwert = 80.0 if temperatur > grenzwert: print("⚠ ALARM: Temperatur zu hoch!") print(f" Aktuell: {temperatur} °C (Limit: {grenzwert} °C)")
if / else – zwei Fälle

Der else-Block läuft immer dann, wenn die if-Bedingung False ist. So deckt man beide Fälle ab.

temp = float(input("Temperatur: ")) if temp > 80: print("ALARM: Überhitzung!") else: print("Temperatur im Normalbereich.")
if / elif / else – mehrere Fälle

Mit elif (else if) kann man beliebig viele Fälle nacheinander prüfen. Python prüft von oben nach unten und führt nur den ersten zutreffenden Block aus.

temp = float(input("Temperatur: ")) if temp > 100: print("KRITISCH: Sofort abschalten!") elif temp > 80: print("WARNUNG: Temperatur erhöht.") elif temp > 60: print("INFO: Betriebstemperatur erreicht.") else: print("Aufwärmphase läuft...")
Reihenfolge beachten

Python führt immer nur den ersten zutreffenden Block aus. Prüfe daher von der engsten zur weitesten Bedingung – hier: zuerst >100, dann >80, dann >60.

Ternary-Operator (optional)

Eine Kurzschreibweise für einfaches if/else in einer Zeile. Nur sinnvoll bei sehr einfachen Bedingungen.

wert_wenn_wahr if bedingung else wert_wenn_falsch
temp = 85 status = "ALARM" if temp > 80 else "OK" print(status) # → ALARM # Entspricht exakt: if temp > 80: status = "ALARM" else: status = "OK"
