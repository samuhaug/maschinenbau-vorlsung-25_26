02

Schleifensteuerung

break – Schleife sofort beenden

break verlässt die Schleife sofort – egal ob die Schleifenbedingung noch True wäre. Der Code nach der Schleife wird normal weiter ausgeführt.

messwerte = [22.1, 24.5, 87.3, 23.8, 22.9] alarm_limit = 80.0 for wert in messwerte: if wert > alarm_limit: print(f"ALARM bei {wert} °C – Stoppe Prüfung!") break # Schleife sofort verlassen print(f"OK: {wert} °C") print("Prüfung abgeschlossen.") # läuft immer # → OK: 22.1 °C # → OK: 24.5 °C # → ALARM bei 87.3 °C – Stoppe Prüfung! # → Prüfung abgeschlossen.
continue – Durchlauf überspringen

continue überspringt den Rest des aktuellen Schleifendurchlaufs und geht direkt zum nächsten Durchlauf. Die Schleife läuft weiter.

messwerte = [22.1, -999, 24.5, -999, 23.8] # -999 bedeutet: Sensor-Fehler, Wert ungültig for wert in messwerte: if wert == -999: print("Sensor-Fehler, überspringe...") continue # Rest überspringen print(f"Gültiger Wert: {wert} °C") # → Gültiger Wert: 22.1 °C # → Sensor-Fehler, überspringe... # → Gültiger Wert: 24.5 °C # → ...
03

Die drei Codeprinzipien

KISS

Keep It Simple, Stupid

Schreibe Code so einfach wie möglich. Wenn du deinen eigenen Code nicht auf Anhieb verstehst, ist er zu komplex.

DRY

Don't Repeat Yourself

Kein Code soll mehrfach vorkommen. Was sich wiederholt, gehört in eine Funktion. Einmal ändern – überall wirksam.

SRP

Single Responsibility

Jede Funktion/jeder Codeabschnitt macht genau eine Sache. Mische nicht Eingabe, Berechnung und Ausgabe in einer Funktion.

## SCHLECHTER Code (verletzt DRY + KISS) print("Sensor 1:", float(input("Temp 1: ")) * 9/5 + 32) print("Sensor 2:", float(input("Temp 2: ")) * 9/5 + 32) print("Sensor 3:", float(input("Temp 3: ")) * 9/5 + 32) ## GUTER Code (DRY: Formel einmal, wiederverwendbar) def celsius_zu_fahrenheit(celsius): return celsius * 9/5 + 32 for i in range(1, 4): celsius = float(input(f"Temp {i}: ")) print(f"Sensor {i}: {celsius_zu_fahrenheit(celsius):.1f} °F")
