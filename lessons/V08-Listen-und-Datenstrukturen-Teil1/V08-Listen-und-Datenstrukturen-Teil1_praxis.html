02

Indexierung in einer Liste

Jedes Element in einer Liste hat einen Index – eine Position, beginnend bei 0. Python erlaubt auch negative Indizes: -1 ist das letzte Element.

messwerte[2] → 87.3 (roter Alarm-Wert)  |  messwerte[-1] → 22.9 (letztes Element)

03

Listen im Detail

Erstellen, Zugreifen, Ändern
# Liste erstellen temperaturen = [22.1, 24.5, 23.8] leere_liste = [] # Zugriff per Index print(temperaturen[0]) # → 22.1 (erstes Element) print(temperaturen[-1]) # → 23.8 (letztes Element) print(len(temperaturen)) # → 3 (Anzahl Elemente) # Element ändern temperaturen[1] = 25.0 # 24.5 → 25.0 # Slicing: Teilliste extrahieren [start:stop] print(temperaturen[0:2]) # → [22.1, 25.0] (stop exklusiv!) print(temperaturen[1:]) # → [25.0, 23.8] (bis Ende)
append() · insert() · pop() · remove()

Die wichtigsten Methoden zum Bearbeiten von Listen:

messwerte = [22.1, 24.5] # append(): Element ans Ende anfügen messwerte.append(23.8) # → [22.1, 24.5, 23.8] # insert(index, wert): an Position einfügen messwerte.insert(0, 20.0) # → [20.0, 22.1, 24.5, 23.8] # pop(): letztes Element entfernen und zurückgeben letzter = messwerte.pop() # letzter = 23.8 # pop(index): Element an Position entfernen erster = messwerte.pop(0) # erster = 20.0 # remove(wert): ersten passenden Wert entfernen messwerte.remove(22.1) # Hilfsfunktionen print(sum(messwerte)) # Summe aller Werte print(max(messwerte)) # Maximalwert print(min(messwerte)) # Minimalwert
Tupel – unveränderliche Liste

Tupel sind wie Listen, können aber nicht verändert werden (immutable). Sie werden mit runden Klammern ( ) erstellt. Ideal für Koordinaten, Farben und andere feste Wertepaare.

# Tupel erstellen koordinate = (0, 2) # Zeile 0, Spalte 2 dhbw_farbe = (226, 0, 26) # RGB: #E2001A sensor_info = ("K-Typ", 23.5, "°C") # Zugriff wie bei Listen print(koordinate[0]) # → 0 print(koordinate[1]) # → 2 # Tuple-Unpacking: Werte direkt auslesen zeile, spalte = koordinate print(f"Zeile: {zeile}, Spalte: {spalte}") # Ändern nicht möglich – führt zu TypeError: # koordinate[0] = 1 ← ❌ TypeError!
2D-Listen – Listen in Listen

Eine Liste kann andere Listen als Elemente enthalten. Damit lassen sich Tabellen und 2D-Spielfelder darstellen.

# 5×5 Spielfeld als 2D-Liste spielfeld = [] for i in range(5): spielfeld.append(["~"] * 5) # 5 Zeilen je 5 "~" # Zugriff: spielfeld[zeile][spalte] print(spielfeld[0][2]) # → "~" (Zeile 0, Spalte 2) # Element setzen spielfeld[0][2] = "X" # Treffer markieren # Messdaten-Tabelle: Zeilen = Messzeitpunkte messreihe = [ [1, 22.1, "OK"], [2, 87.3, "ALARM"], [3, 23.8, "OK"], ] for nr, temp, status in messreihe: print(f"Messung {nr}: {temp:.1f} °C – {status}")

Im Schiffe-Versenken-Spiel

Spielfeld als 2D-Liste · Schiff als Tupel-Liste

Das Herzstück des Spiels

# Spielfeld: 2D-Liste, 5×5, gefüllt mit "~" (Wasser) spielfeld = [] for i in range(5): spielfeld.append(["~"] * 5) # Schiff: Liste von Koordinaten-Tupeln schiff = [(0, 1), (0, 2), (0, 3)] # Treffer markieren zeile, spalte = 0, 2 if (zeile, spalte) in schiff: spielfeld[zeile][spalte] = "X" # Treffer else: spielfeld[zeile][spalte] = "O" # Wasser
