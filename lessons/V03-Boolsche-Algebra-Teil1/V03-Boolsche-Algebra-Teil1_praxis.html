02

Die vier Grunddatentypen

Python hat vier Grundtypen, die im Maschinenbau ständig vorkommen. Jeder Typ hat eine bestimmte Verwendung:

int Ganzzahl

Messung-Nummern, Anzahl Teile, Schleifenzähler

messung_nr = 42
float Dezimalzahl

Temperaturen, Drücke, Drehzahlen, Koordinaten

temperatur = 23.5
str Text

Sensor-Namen, Einheiten, Benutzereingaben

einheit = "°C"
bool Wahrheitswert

Schaltzustände, Bedingungen, Alarmflags

alarm = True
03

Funktionen im Detail

type()  –  Typ abfragen

Gibt den Datentyp einer Variable zurück. Nützlich zum Debuggen, wenn unerwartete Typen auftauchen.

temperatur = 23.5 messung_nr = 7 sensor_name = "K-Typ" sensor_aktiv = True print(type(temperatur)) # → <class 'float'> print(type(messung_nr)) # → <class 'int'> print(type(sensor_name)) # → <class 'str'> print(type(sensor_aktiv)) # → <class 'bool'> # input() liefert IMMER einen str! eingabe = input("Wert: ") print(type(eingabe)) # → <class 'str'>
int()  ·  float()  ·  str()  –  Umwandeln

Umwandlungsfunktionen konvertieren Werte von einem Typ in einen anderen. Das ist besonders wichtig, da input() immer Text zurückgibt.

int(wert)  ·  float(wert)  ·  str(wert)
# str → int / float (für Berechnungen) eingabe = "42" als_int = int(eingabe) # → 42 (Ganzzahl) als_float = float(eingabe) # → 42.0 (Dezimalzahl) # float → int (Nachkommastellen werden abgeschnitten!) temp = 23.9 print(int(temp)) # → 23 (NICHT gerundet!) # int / float → str (für Textausgaben) rpm = 1450 als_str = str(rpm) # → "1450" print("Drehzahl: " + als_str + " RPM") # Typisches Muster: input → float → Berechnung wert = float(input("Temperatur: ")) print(f"In Kelvin: {wert + 273.15:.2f} K")
⚠ Häufiger Fehler: TypeError

Du kannst einen String nicht mit einer Zahl addieren. "23.5" + 10 ergibt einen Fehler. Lösung: erst umwandeln mit float("23.5") + 10.

bool()  –  Wahrheitswert

Boolesche Werte (True / False) repräsentieren Schaltzustände. In Maschinensystemen: Ventil offen/zu, Sensor aktiv/inaktiv, Grenzwert überschritten/nicht überschritten.

motor_aktiv = True alarm_aktiv = False notaus = False print(type(motor_aktiv)) # → <class 'bool'> # Umwandlung von Zahlen in bool print(bool(0)) # → False (Null ist immer False) print(bool(1)) # → True print(bool(-5)) # → True (alle Zahlen ≠ 0 sind True) print(bool("")) # → False (leerer String ist False) print(bool("Hallo")) # → True
04

Im Schiffe-Versenken-Spiel

Der wichtigste Anwendungsfall: die Benutzereingabe "A3" muss in Koordinaten umgewandelt werden.

🚢

Koordinaten-Umwandlung

str → int für Zeilenindex

eingabe = input("Schuss (z.B. A3): ") # Typ: str # Buchstabe → Spalten-Index (0-4) spalte = "ABCDE".index(eingabe[0]) # "A" → 0 # Ziffernzeichen → Zeilen-Index (0-4) zeile = int(eingabe[1]) - 1 # "3" → 2 print(type(eingabe[1])) # → <class 'str'> (Zeichen, keine Zahl!) print(type(zeile)) # → <class 'int'> (nach Umwandlung)
