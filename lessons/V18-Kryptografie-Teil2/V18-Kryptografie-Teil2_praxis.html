02

Wie Hashing funktioniert

Eigenschaft

Eine minimale Änderung an der Datei (ein Bit!) ergibt einen komplett anderen Hashwert. Zwei Dateien mit gleichem Hashwert sind garantiert identisch.

03

hashlib im Detail

hashlib – Text hashen

hashlib ist Teil der Python-Standardbibliothek. Wichtig: die Funktion erwartet bytes, keine Strings. Texte müssen mit .encode() umgewandelt werden.

import hashlib # Text hashen text = "Messwert: 23.5 °C" als_bytes = text.encode("utf-8") # str → bytes (Pflicht!) hashwert = hashlib.sha256(als_bytes).hexdigest() print(hashwert) # → "a3f1c2d9e8b7..." (64 Hexadezimalzeichen) # Gleicher Text → immer gleicher Hash h1 = hashlib.sha256(b"Test").hexdigest() h2 = hashlib.sha256(b"Test").hexdigest() print(h1 == h2) # → True (immer!) # Minimale Änderung → völlig anderer Hash h3 = hashlib.sha256(b"test").hexdigest() # Kleinbuchstabe! print(h1 == h3) # → False (komplett anders)
Datei-Integrität prüfen

Praktischer Anwendungsfall: nach dem Herunterladen einer Firmware oder Messdaten-Datei prüfen, ob die Datei vollständig und unverändert übertragen wurde.

import hashlib def datei_hash(dateipfad): sha256 = hashlib.sha256() with open(dateipfad, "rb") as f: # "rb" = binär lesen! while True: block = f.read(65536) # 64 KB-Blöcke if not block: break sha256.update(block) return sha256.hexdigest() # Bekannter Referenz-Hash (z.B. vom Hersteller) referenz = "abc123def456..." aktuell = datei_hash("firmware_v2.bin") if aktuell == referenz: print("✅ Datei unverändert – sicher!") else: print("⚠ WARNUNG: Datei wurde verändert!")
str vs. bytes – der Unterschied

Hashlib arbeitet immer mit bytes. Der Unterschied ist wichtig zu kennen.

# str (Text): in Python intern Unicode text = "Hallo" print(type(text)) # → <class 'str'> # bytes: Rohdaten (Zahlen 0-255) roh = b"Hallo" # b-Prefix = bytes print(type(roh)) # → <class 'bytes'> # Umwandlung str ↔ bytes als_bytes = "Hallo".encode("utf-8") # str → bytes als_str = b"Hallo".decode("utf-8") # bytes → str # Dateien im binären Modus lesen with open("bild.png", "rb") as f: # "rb" = read binary inhalt = f.read() # → bytes

Spielstand-Integrität prüfen

Wurde der Spielstand manipuliert?

import hashlib def spielstand_hash(schüsse, treffer): inhalt = f"{schüsse},{treffer}" return hashlib.sha256(inhalt.encode()).hexdigest() # Beim Speichern: Hash mitspeichern schuesse, treffer = 12, 7 pruefsumme = spielstand_hash(schuesse, treffer) # Beim Laden: Hash verifizieren if spielstand_hash(schuesse, treffer) == pruefsumme: print("Spielstand authentisch.") else: print("⚠ Spielstand wurde manipuliert!")
