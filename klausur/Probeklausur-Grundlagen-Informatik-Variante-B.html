B1 · Python in einen Programmablaufplan (PAP) übertragen

12 Punkte

Zeichnen Sie zum folgenden Python-Programm einen vollständigen Programmablaufplan (PAP) nach DIN 66001. Achten Sie auf die korrekten Symbole für Start/Ende, Ein-/Ausgabe, Verarbeitung und Verzweigung sowie auf die richtige Darstellung der Schleife mit Zählvariable und der inneren Verzweigung.

zahlen = [4, 9, 2, 7, 5]
groesster = zahlen[0]
i = 1
while i < len(zahlen):
    if zahlen[i] > groesster:
        groesster = zahlen[i]
    i = i + 1
print(groesster)

B3 · Logikfehler finden und korrigieren

11 Punkte

In den folgenden Programmen ist jeweils ein Logikfehler enthalten (der Code lässt sich fehlerfrei ausführen, liefert aber nicht das gewünschte Ergebnis). Benennen Sie den Fehler, erklären Sie kurz, warum das Ergebnis falsch ist, und geben Sie die korrigierte Zeile an.

a) Die Funktion soll die Summe aller Zahlen der Liste zurückgeben. (6 P.)

def summe_liste(zahlen):
    summe = 0
    for z in zahlen:
        summe = summe + z
        return summe

b) Die Funktion soll True liefern, wenn der gesuchte Wert in der Liste vorkommt. (5 P.)

def ist_enthalten(liste, gesucht):
    for element in liste:
        if element == gesucht:
            return True
        else:
            return False
