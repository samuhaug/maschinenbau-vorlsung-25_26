B1 · Python in einen Programmablaufplan (PAP) übertragen

12 Punkte

Zeichnen Sie zum folgenden Python-Programm einen vollständigen Programmablaufplan (PAP) nach DIN 66001. Achten Sie auf die korrekten Symbole für Start/Ende, Ein-/Ausgabe, Verarbeitung und Verzweigung sowie auf die richtige Behandlung der Schleife mit Abbruchbedingung (Sentinel) und der inneren Verzweigung.

gerade = 0
ungerade = 0
zahl = int(input("Zahl (0 = Ende): "))
while zahl != 0:
    if zahl % 2 == 0:
        gerade = gerade + 1
    else:
        ungerade = ungerade + 1
    zahl = int(input("Zahl (0 = Ende): "))
print(gerade, ungerade)

B3 · Logikfehler finden und korrigieren

11 Punkte

In den folgenden Programmen ist jeweils ein Logikfehler enthalten (der Code lässt sich fehlerfrei ausführen, liefert aber nicht immer das gewünschte Ergebnis). Benennen Sie den Fehler, erklären Sie kurz, warum das Ergebnis falsch ist, und geben Sie die korrigierte Zeile an.

a) Die Funktion soll den größten Wert der Liste zurückgeben. Für welche Eingaben liefert sie ein falsches Ergebnis? (6 P.)

def groesster_wert(zahlen):
    groesster = 0
    for z in zahlen:
        if z > groesster:
            groesster = z
    return groesster

b) Die Funktion soll True liefern, wenn die Liste mindestens einen doppelten Wert enthält. (5 P.)

def enthaelt_doppelte(liste):
    for i in range(len(liste)):
        for j in range(len(liste)):
            if liste[i] == liste[j]:
                return True
    return False
